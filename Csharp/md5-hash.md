# MD5 File Hashing

## MD5 Byte Array Hash with C#
```cs
public static string MD5(byte[] bytes)
{
    var md5 = System.Security.Cryptography.MD5.Create();
    return string.Join("", md5.ComputeHash(bytes).Select(x => x.ToString("x2")).ToArray());
}
```

## MD5 File Hash with C#

```cs
string GetMD5Hash(string filePath)
{
    var md5 = System.Security.Cryptography.MD5.Create();
    using var stream = File.OpenRead(filePath);
    return string.Join("", md5.ComputeHash(stream).Select(x => x.ToString("x2")));
}
```

```cs
var md5 = System.Security.Cryptography.MD5.Create();
string[] imageFilePaths = System.IO.Directory.GetFiles($"./", "*.png");
foreach (string filePath in imageFilePaths)
{
    string hashString = "";
    using (var stream = System.IO.File.OpenRead(filePath))
    {
        byte[] hashBytes = md5.ComputeHash(stream);
        for (int i = 0; i < hashBytes.Length; i++)
            hashString += hashBytes[i].ToString("X2");
    }
    Console.WriteLine($"{hashString} {System.IO.Path.GetFileName(filePath)}");
}
```
